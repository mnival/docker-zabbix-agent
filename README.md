Docker debian-zabbix-agent
============

Configuration Docker with Debian Stable and package : zabbix-agent

Quick Start
===========
    docker run -d -p 10050 --name zabbix-agent mnival/debian-zabbix-agent

Interfaces
===========

Ports
-------

* 10050 -- zabbix-agent

Volumes
-------

* /etc/zabbix/zabbix_agentd.d/
